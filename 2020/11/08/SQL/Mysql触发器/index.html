Mysql触发器

✦ 2020-11-08 ☛ CS ， SQL

一 概述

  1. 事先为某张表绑定好一段代码 ,当表中的某些内容发生改变的时候（增删改）系统会自动触发代码执行
  2. 事件类型
    • 增 insert
    • 删 delete
    • 改 update
  3. 触发时间
    • 前 before
    • 后 after
  4. 一张表最多能有 6 种触发器（事件类型 + 触发事件）
    1. 增 前/后
    2. 删 前/后
    3. 改 前/后
  5. 触发对象
    • 表中的每一条记录（行）
  6. new/old
    • 可以用来获取新添加的或者删除的记录

三 触发器管理

命令 说明
show triggers 查看所有触发器

drop trigger 触发器名 删除触发器

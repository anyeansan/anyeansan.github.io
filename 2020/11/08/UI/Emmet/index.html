✦ 2020-11-08 ☛ CS ， UI

一 概述

  1. Emmet (前身为 Zen Coding) 是一个能大幅度提高前端开发效率的一个工具
  2. 语法默认以 tab 结束

二 语法

2.1 Html

1 元素

  1. 快速生成Html模板

  2. 使用元素名称生成html标签：元素名

  3. 数量：元素名*num

  4. 自增符号：$

2 层级

  1. 父子：父元素>子元素

  2. 兄弟：兄元素+弟元素

  3. 注意：会先算兄弟后算父子

  4. 优先级：()

3 属性

  1. 创建带 id 的标签：元素名#id

  2. 创建带 class 的标签：元素名.class

  3. 创建带属性的标签：元素名[属性名='属性值']

  4. 创建带内容的标签：元素名{内容}

  5. 隐式标签

MongoDB

✦ 2020-03-30 ☛ DB ， MongoDB

一 安装

  1. 下载解压，并修改属主属组
  2. 关闭 Linux 大页内存 THP（transparent huge pages）
    • https://docs.mongodb.com/v3.6/tutorial/transparent-huge-pages/
  3. 创建所需用户和用户组
  4. 创建存储目录并修改属主属组
    1. data directory
    2. log file
    3. conf directory
  5. 设置用户环境变量
    1. 切换到创建用户
    2. 修改用户目录下 .bash_profile
    3. source .bash_profile

二 配置

  • 在指定的 conf directory 下创建配置文件

三 启动与关闭 mongod

3.1 启动

  1. mongod -f 配置文件
    • 启动

3.2 关闭

  1. mongod -f 配置文件 --shutdown
    • 关闭
  2. db.shutdownServer()
  3. kill -2 <mongod process ID>

四 登录与退出

  1. mongo -u -p ip/验证库
  2. exit

五 概述

5.1 逻辑结构

概念 说明
database 库
collection 表（集合）
document 文档（数据行）
field 字段
index 索引
primary key 主键，MongoDB自动将 _id 字段设置为主键
* 文档数据都是 BSON 格式。
* BSON 是一种类似 JSON 的二进制形式的存储格式，是 Binary JSON 的简称

5.2 默认库

  1. test
    • 登录时默认所在库
  2. admin
    • 系统预留库，MongoDB系统管理库
  3. local
    • 本地预留库，存储关键日
  4. config
    • MongoDB配置信息库

六 操作

6.1 库操作

1 帮助

  1. db.help()
    • 帮助
  2. db.[tab][tab]
    • 两次 tab

2 查

  1. show dbs / show databases
  2. db
    • 相当于 select database()
  3. db.status()

3 增

  • use db_name
    • 若存在，切换到数据库，
    • 若不存在，则创建并切换（注意库中存有数据才会显示在数据库列表中）

4 删

  • db.dropDatabase()

6.2 集合操作

1 查

  • show collections / show tables

2 增

  • db.createCollection(name, options)
    • name
      • 集合名称
    • options
      1. capped
        1. false ：默认
        2. true ：创建固定大小的集合，当达到最大值时，它会自动覆盖最早的文档，必须指定 size 参数
      2. size
        • 为固定集合指定一个最大值（KB）
      3. max
        • 指定固定集合中包含文档的最大数量
      4. autoIndexId
        1. false ：默认
        2. true ：自动在 _id 字段创建索引

3 删

  • db.collection.drop()

6.3 文档操作

1 增

  1. db.collection_name.insertOne()
    • 如果指定 collection 不存在，则会创建
  2. db.collection_name.insertMany()

2 删

  1. db.collection_name.deleteOne()
  2. db.collection_name.deleteMany()

4 改

  1. db.collection_name.updateOne()
  2. db.collection_name.updateMany()
  3. db.collection_name.replaceOne()

4 查

1 语法

  • db.collection_name.find()[.limit()][.pretty()]
    • 查询所有
    • .pretty()
      • 以 json 格式显示结果

2 比较

格式 说明
{key:value} 等于
{key:{$ne:value}} 不等于
{key:{$lt:value}} 小于
{key:{$lte:value}} 小于等于
{key:{$gt:value}} 大于
{key:{$gte:value}} 大于等于

4 or
